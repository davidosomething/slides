Install WordPress

We'll be installing from your backup.

  • Step 1: Copy your wordpress folder to the right place.
          /Applications/MAMP/htdocs
  • Step 2: Give WordPress a database to use.
    Open up the wp-config.php file in your wordpress folder to see what credentials you use last time.
  • Step 3: Restore your database
          1. go to phpmyadmin
          2. open your database
          3. click import
          4. browse for the sql/zip/gz/bz file you backed up to and click the button to restore the database

Exploring other parts of WordPress

First switch to the twenty-eleven theme

Dashboard

The Dashboard is the homepage for the administration panel.

It has widgets on it, like your sidebars:

  • Welcome
  • Right Now -- some basic statistics about your blog
  • QuickPress -- quickly and easily add posts
  • Recent Drafts -- posts you haven't finished writing
  • Recent Comments
  • Incoming Links
  • Plugins
  • WordPress Blog
  • Other WordPress News

Posts (review)

Posts are best suited for content that is updated regularly:

  • Blog posts
  • Daily/weekly/regular news posts
  • Site updates
  • Journal/Diary

Typically older posts are not shown on the site unless you have pagination set up.

Media

This is where your uploaded images are collected.

You can edit the image meta data here.

You can delete or change images here.

Links

This section adds/edits links to the blogroll.

The blogroll is just a list of links that can be added to your site as a widget.

Pages

Pages are best used for content that is always on the site such as:

  • Static home page
  • About me page
  • Contact us page
  • Privacy Policy page

Comments

You can manage — approve and delete — comments in this section.

Posts have comments enabled by default. Pages don't, but you can enable them.

Appearance

This is where you manage and install themes, manage widgets (but not install), and manage menus and theme options.

Plugins

Add and install plugins here.

Plugins can change WordPress in ANY way. Some examples:

  • Adding widgets.
  • Automatically creating posts and pages.
  • Allowing you to login via Facebook or Twitter.
  • Changing the commenting system.
  • Adding advertisements.
  • Adding analytics for tracking visitors.
  • Theming the dashboard.
  • Finding broken links.
  • Automatically tagging posts.
  • Automatically tweeting posts.
  • much much more...

Users

Add, delete, and edit user profiles here.

You can prevent people from commenting on your site unless they have an account.

You can create accounts for your friends or co-workers to create posts and run a team or corporate blog.

You can password protect posts and pages so only people with accounts can read them.

Tools

Export and import posts here. You can move posts to another blog, or use this for a cheap backup system (but a full MySQL export is better!).

Press This is a bookmarklet that opens a popup window to save a link as a post in your blog. This turns WordPress into something like tumblr so you collect links and photos.

Settings

The name of your site, description, URL, and timezone are here.

You can change your site's homepage page/posts using the Reading Settings.

You can turn on/off commenting in the Discussion settings.

You can change the size of uploaded images in the Media section.

You can prevent google from indexing your site in the Privacy settings.

You can make your URLs look nicer in the Permalink Settings

Usually plugins add their settings pages here, or into Tools.

WordPress Mobile Apps

Mobile apps

Download them here.

They allow you to view and edit your posts, pages, and comments from your phone.

So if you see something... blog about it from anywhere

More WordPress theming

More WordPress theming

<?php get_header(); ?>

This includes the header.php file.

<?php get_footer(); ?>

This includes the footer.php file.

<?php get_sidebar(); ?>

This includes the sidebar.php file.

<?php get_template_part('something'); ?>

This includes something.php in your theme.

<?php echo get_stylesheet_directory_uri(); ?>

This outputs the theme folder URL.

More WordPress theming

post.php - this is the template for a post.

page.php - this is the template for a page.

page-xyz.php - this is the template for the page with slug 'xyz'.

These works just like the index page.

For more options, see the Template Hierarchy diagram.
