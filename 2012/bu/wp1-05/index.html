Install WordPress

We'll be installing from your backup.

  • Step 1: Copy your wordpress folder to the right place.
          /Applications/MAMP/htdocs
  • Step 2: Give WordPress a database to use.
    Open up the wp-config.php file in your wordpress folder to see what credentials you use last time.
  • Step 3: Restore your database
          1. go to phpmyadmin
          2. open your database
          3. click import
          4. browse for the sql/zip/gz/bz file you backed up to and click the button to restore the database

Vocabulary review

  • <span class="awesome">I'm awesome!</span> — This is an HTML element. It has tags, content, and other hidden properties.
  • <span class="awesome">I'm awesome!</span> — This is an HTML start tag.
  • <span class="awesome">I'm awesome!</span> — This is the tag's name.
  • <span class="awesome">I'm awesome!</span> — This is an attribute.
  • <span class="awesome">I'm awesome!</span> — This is the attribute's value.
  • <span class="awesome">I'm awesome!</span> — This is the contents of the HTML element.
  • <span class="awesome">I'm awesome!</span> — This is the end tag for the span element.

Vocabulary review

  • nav a { text-decoration: none; color: blue; } — This is a CSS rule.
  • nav a { text-decoration: none; color: blue; } — This is the rule's selector.
  • nav a { text-decoration: none; color: blue; } — These are the style declarations for the rule.
  • nav a { text-decoration: none; color: blue; } — This is a style declaration's property.
  • nav a { text-decoration: none; color: blue; } — This is that property's value.

Vocabulary review

  • <?php the_title(); ?> — This is a PHP block.
  • <?php the_title(); ?> — This is a PHP start tag.
  • <?php the_title(); ?> — This is a function statement. It "calls" the function named "the_title".
  • <?php the_title(); ?> — This is a PHP end tag.

More WordPress theming

<?php get_header(); ?>

This includes the header.php file.

<?php get_footer(); ?>

This includes the footer.php file.

<?php get_sidebar(); ?>

This includes the sidebar.php file.

<?php get_template_part('something'); ?>

This includes something.php in your theme.

<?php echo get_stylesheet_directory_uri(); ?>

This outputs the theme folder URL. Use this for images that aren't backgrounds.

More WordPress theming

single.php - this is the template for a post.
